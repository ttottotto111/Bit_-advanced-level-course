[ 오목 웹 서비스 구현 ]


1) 프로젝트 생성

  ASP.NET 웹 서비스 템플릿 생성

  위치 : http://localhost/OmokAIService


  - 기본적으로 코드 비하인드가 적용됨(Code Behind)

  - 오목 웹 서비스가 제공해야 하는 것들을 Service.asmx.cs 파일에 작성함.


2) 오목 웹 서비스 객체 Service1

   Service1 클래스에 오목 알고리즘과 관련된 어플리케이션 로직을 작성

   - C#으로 작성하는 것과 동일

   - 차이점 : [WebMethod]와 같은 추가적인 어트리뷰트를 요구함

   => 01_웹서비스asmx 확인해 볼것


3) 오목 웹 서비스 클라이언트

   - 폼을 나타내기 위한 클래스

   - 오목 웹 서비스와 통신을 하는 클래스 


   OmokForm : 오목판을 화면에 표시하고, 사용자의 입력을 받아서 오목판에 돌을 둘 수 
	있는 폼

   AboutForm : 오목 게임에 대한 정보를 표시하는 폼

   OmokJumsInfoForm : 현재 오목판에 돌이 놓여진 상태를 수치화해서 보여주고,
		사용자가 직접 편집할 수 있도록 해줌


   StartForm : 오목 게임을 처음 시작하거나, "새게임" 메뉴를 클릭하면 나타나는 폼

	사용자가 먼저 돌을 둘 것인지, 컴퓨터가 먼저 돌을 둘 것인지를 결정

   OmokData : 현재 오목판의 상태를 저장하고 있는 클래스

       오목 웹 서비스가 제공하는 객체를 가지고 있다.
